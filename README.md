 ## How 
 - Clone the repo in Ubuntu OS
 - In treminal
 - Make sure you have the libfreetype6-dev library installed
  ```
 sudo apt update
 sudo apt install libfreetype6-dev
 ```
 Then you can run the program using:
 
 ```
 cd IPC-under-linux-Project2
 make
 ./parent
 ```
